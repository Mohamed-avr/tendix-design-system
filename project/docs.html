Tendix Design System

Reusable components for the Tendix university attendance platform. Available for React (web) and React Native / Expo via React Native Paper. All components share the same token vocabulary — swap the platform, keep the brand.

Web · ReactHTML + JSX
// 1. link styles
<link rel="stylesheet" href="/styles.css">
// 2. load bundle (after React 18)
<script src="/_ds_bundle.js"></script>
// 3. use
const { Button } = window.DesignSystem_83ace9;
React Native · ExpoRN Paper
// install
npx expo install react-native-paper
// wrap app
import { studentTheme } from './react-native/theme';
<PaperProvider theme={studentTheme}>
  <App />
</PaperProvider>
Installation

The web system needs no package manager — just two files. For React Native, install React Native Paper and copy the react-native/ folder into your project.

Web
<!-- In your HTML <head> -->
<link rel="stylesheet" href="/styles.css" />

<!-- React 18 (use .production.min.js in prod) -->
<script src="https://unpkg.com/react@18.3.1/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@18.3.1/umd/react-dom.development.js"></script>

<!-- Component bundle -->
<script src="/_ds_bundle.js"></script>

<script type="text/babel">
const { Button, Card, Badge, StatusBadge } = window.DesignSystem_83ace9;
</script>
React Native / Expo
# Install peer dependencies
npx expo install react-native-paper @expo/vector-icons react-native-safe-area-context

# Install Urbanist + Inter fonts
npx expo install @expo-google-fonts/urbanist @expo-google-fonts/inter expo-font

# Copy the react-native/ folder into your project, then:
import { studentTheme } from './react-native/theme';
import { PaperProvider } from 'react-native-paper';

export default function App() {
  return (
    <PaperProvider theme={studentTheme}>
      <YourApp />
    </PaperProvider>
  );
}
Theming

Tendix is role-based. Each user role has a distinct accent colour. Set data-role on a container (web) or use getThemeForRole() (RN) to retheme that subtree.

Web — data-role attribute
student
profs
admin 
<div data-role="student"> // amber --tx-accent
  <Button color="accent">Find the class</Button>
</div>

<div data-role="profs">   // blue --tx-accent
  <Button color="accent">Mark attendance</Button>
</div>

<div data-role="admin">   // green --tx-accent
  <Button color="accent">Approve request</Button>
</div>
React Native — getThemeForRole
import { getThemeForRole } from './react-native/theme';
import { PaperProvider } from 'react-native-paper';

// Wrap per-screen or per-app
export function StudentScreen() {
  return (
    <PaperProvider theme={getThemeForRole('student')}>
      <HomeView />
    </PaperProvider>
  );
}

// Or per-component — pass role prop
<TxButton role="admin" color="accent">Approve</TxButton>

Card

The system's primary surface — white, 24px radius, soft shadow, faint hairline. inset gives the sunken grey row variant.

// Default card
<Card>
  <h3>Software Engineering</h3>
  <p>L3 · Module 6</p>
</Card>

// Inset (sunken grey row)
<Card inset padding={12}>
  <ClassRow title="TD 01" />
</Card>

// Interactive (hover lift)
<Card interactive onClick={() => navigate('detail')}>
  <ClassCard />
</Card>

BottomNav

Tab bar for mobile-style navigation. Active item expands to an ink pill; inactive items show icon + label in muted colour.

import { TxBottomNav } from './react-native';
import { useState } from 'react';

export function StudentLayout() {
  const [tab, setTab] = useState('home');
  return (
    <SafeAreaView style={{ flex: 1 }}>
      <ScreenContent tab={tab} />
      <TxBottomNav value={tab} onChange={setTab} />
    </SafeAreaView>
  );
}

// React Navigation tab bar adapter
// In your Tab.Navigator:
tabBar={(props) => (
  <TxBottomNav
    value={props.state.routeNames[props.state.index]}
    onChange={(v) => props.navigation.navigate(v)}
    items={props.state.routeNames.map((name) => ({
      value: name,
      label: name,
      icon: iconMap[name],
    }))}
  />
)}
