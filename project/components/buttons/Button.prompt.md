One-tap action control — pillised and ink-filled by default, matching Tendix's soft, fully-rounded house style.

```jsx
<Button>Find The Class</Button>
<Button color="accent" variant="soft">Justify your absence</Button>
<Button variant="outlined" size="sm">February</Button>
<Button color="primary" fullWidth size="lg">Next</Button>
```

Variants: `contained` (default), `outlined`, `soft`, `text`. Colors: `primary` (ink/near-black — the default CTA), `accent` (role theme: amber/blue/green), `error`, `success`. Sizes `sm | md | lg`. `pill` defaults on; pass `pill={false}` for 8px-radius corners. Use `startIcon` / `endIcon` with the `Icon` component.
