Quick start

Two files, zero build step.

Link one stylesheet, load one bundle, and every component is ready under a single global namespace.

1

Link the stylesheet

<!-- tokens + fonts -->
<link rel="stylesheet"
  href="/styles.css">
2

Load the bundle

<!-- React 18 first -->
<script src="react.js"></script>
<script src="/_ds_bundle.js"></script>
3

Use components

const { Button, Logo,
  StatusBadge, Icon }
  = window.DesignSystem_83ace9;

Role theming — one attribute

<!-- set data-role on any container to retheme --tx-accent -->
<div data-role="student"> …amber accent… </div>
<div data-role="profs">   …blue accent…  </div>
<div data-role="admin">   …green accent… </div>
